Detailed Description

Methods for manipulating the console.

Useful for cursor placement and visibility, foreground and background colors, and clearing part or all of the screen.

Constructor & Destructor Documentation

◆ ~console()

virtual cpen333::console::~console ( )
inlinevirtual

Destructor, currently does nothing.

Note that once the console instance is destructed, any modified console attributes will remain in effect. It is highly recommended to call clear_all() prior to terminating your program.

◆ clear_all()

void cpen333::console::clear_all ( )
inline

Clears display and resets all console attributes.

Resets all console attributes, including foreground and background colours as well as cursor visibility, and clears the contents of the entire display. The cursor's position is set to the top-left position. This method should be called prior to program termination to prevent modified console attributes from persisting beyond the lifetime of the process.
