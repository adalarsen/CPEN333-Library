Detailed Description

A shared mutex implementation with balanced priorities.

A more fair shared mutex, access is granted in batches: 1 writer, batch of readers, 1 writer, batch of readers Based on the alternating method described here: http://www.tools-of-computing.com/tc/CS/Monitors/AlternatingRW.htm
