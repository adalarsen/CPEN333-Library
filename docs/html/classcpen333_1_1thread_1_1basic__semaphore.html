Detailed Description

template<typename Mutex, typename CondVar>
class cpen333::thread::basic_semaphore< Mutex, CondVar >

A local semaphore synchronization primitive.

Used to protect access to a counted resource shared by multiple threads. Contains an integer whose value is never allowed to fall below zero. There are two main supported actions: wait(), which decrements the internal value, and notify() which increments the value. If the value of the semaphore is zero, then wait() will cause the thread to block until the value becomes greater than zero.

This implementation has no explicit maximum value.
